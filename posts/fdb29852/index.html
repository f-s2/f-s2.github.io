Hexo常用命令

基本命令

  1. hexo new post “post-name” # 新建博客
  2. hexo clean # 清除缓存
  3. hexo generate # 生成静态页至 public 目录
  4. hexo server # 开启本地预览（默认为 4000，ctrl + c 关闭）
  5. hexo deploy # 部署到 github
  6. hexo version # 查看 hexo 版本
  7. hexo help # 查看帮助

缩写

  1. hexo n === hexo new
  2. hexo g === hexo generate
  3. hexo s === hexo server
  4. hexo d === hexo deploy

组合使用

  1. hexo s -g # 生成并本地预览
  2. hexo d -g # 生成并上传
